## 🔧 快速命令參考

```bash
# ========== 開發 ==========
npm run dev                 # 在瀏覽器中開發（最快）
npm run build:mobile        # 構建行動應用
npm run cap:sync            # 同步到原生平台

# ========== 開啟 IDE ==========
npm run cap:android         # 開啟 Android Studio
npm run cap:ios             # 開啟 Xcode (僅 Mac)

# ========== 維護 ==========
npm run cap:update          # 更新 Capacitor

# ========== 一鍵工具 ==========
build-mobile.bat            # Windows 一鍵構建
```

## 📋 開發流程建議

### 第一次使用？按照這個順序：

1. ✅ **閱讀 `CAPACITOR_SETUP_完成.md`**
   - 了解已完成的設置
   - 查看可用命令
   - 了解專案結構

2. ✅ **閱讀 `CAPACITOR_QUICKSTART.md`**
   - 5 分鐘快速開始
   - 測試應用運行

3. ✅ **準備資源** (`resources/README.md`)
   - 準備應用圖標
   - 準備啟動畫面

4. ✅ **在實體裝置測試**
   - Android 手機
   - iPhone (如有)

5. ✅ **準備上架** (`APP_STORE_CHECKLIST.md`)
   - 按檢查清單逐項完成
   - 參考 `MOBILE_APP_SETUP.md` 獲取詳細說明

## 💡 常見情境

### 我想在手機上看到我的應用
```bash
build-mobile.bat
npm run cap:android
# 在 Android Studio 中點擊運行
```
👉 詳見：`CAPACITOR_QUICKSTART.md`

### 我想更改應用圖標
1. 準備 `resources/icon.png` (1024x1024)
2. 執行 `npx @capacitor/assets generate`
3. 執行 `npm run cap:sync`

👉 詳見：`resources/README.md`

### 我想上架到 Google Play
1. 打開 `APP_STORE_CHECKLIST.md`
2. 按照 Android 章節的檢查清單執行
3. 需要詳細說明時參考 `MOBILE_APP_SETUP.md`

### 我想上架到 App Store
1. 打開 `APP_STORE_CHECKLIST.md`
2. 按照 iOS 章節的檢查清單執行
3. 需要詳細說明時參考 `MOBILE_APP_SETUP.md`

### 我遇到錯誤
1. 先查看 `CAPACITOR_QUICKSTART.md` 的常見問題
2. 再查看 `MOBILE_APP_SETUP.md` 的故障排除
3. 搜尋 Capacitor 官方文檔
