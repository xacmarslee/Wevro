# Wevro 應用截圖指南

## 📸 需要準備的截圖

### Google Play（Android）
- **數量：** 2-8 張（建議 6 張）
- **尺寸：** 至少 1080x1920 (9:16)
- **格式：** PNG 或 JPG

### App Store（iOS）
需要三種尺寸：
- **6.7" iPhone：** 1290x2796（必需）
- **6.5" iPhone：** 1242x2688（必需）
- **5.5" iPhone：** 1242x2208（必需）

---

## 🎯 建議的 6 張截圖內容

### 截圖 1：歡迎/登入畫面 ✨
**目的：** 展示專業的首次印象

**拍攝步驟：**
1. 開啟應用（登出狀態）
2. 確保顯示登入/註冊選項
3. 確保 Logo 和標語清晰可見

**文案疊加（可選）：**
```
AI 英文學習革命
心智圖 + 智慧字卡
```

---

### 截圖 2：心智圖編輯器 🧠
**目的：** 展示核心功能 - 視覺化學習

**拍攝步驟：**
1. 建立一個心智圖（例如：happy）
2. 展開 2-3 層（同義詞、反義詞、衍生詞）
3. 確保節點排列整齊美觀
4. 建議用亮色主題拍攝

**範例單字：**
- happy → joyful, cheerful, delighted
- run → sprint, jog, dash
- beautiful → gorgeous, stunning, lovely

**文案疊加：**
```
視覺化記憶
效果提升 3 倍
```

---

### 截圖 3：字卡列表 📚
**目的：** 展示字卡組織功能

**拍攝步驟：**
1. 建立 3-4 個字卡組
2. 每個字卡組命名清楚（例如：TOEFL 必備、商業英文）
3. 顯示字卡數量

**文案疊加：**
```
智慧字卡組
科學化管理
```

---

### 截圖 4：字卡練習 🎴
**目的：** 展示學習功能

**拍攝步驟：**
1. 開始字卡練習
2. 拍攝顯示單字卡片的畫面
3. 確保顯示「已知/未知」按鈕

**範例字卡：**
```
正面：exquisite
背面：adj. 精緻的；優美的
```

**文案疊加：**
```
間隔重複系統
記憶更持久
```

---

### 截圖 5：例句生成 💬
**目的：** 展示 AI 功能

**拍攝步驟：**
1. 在 Query 頁面點擊「例句」標籤
2. 輸入一個單字（例如：accomplish）
3. 點擊生成
4. 確保顯示多個例句和中英對照

**文案疊加：**
```
AI 即時生成
自然真實的例句
```

---

### 截圖 6：同義詞比較 🔍
**目的：** 展示進階 AI 功能

**拍攝步驟：**
1. 在 Query 頁面點擊「同義詞」標籤
2. 輸入一個單字（例如：happy）
3. 點擊生成
4. 確保顯示 3-5 個同義詞和差異說明

**文案疊加：**
```
深度分析差異
精準表達想法
```

## 📱 如何拍攝截圖

### Android（推薦先用這個）

**使用模擬器：**
```bash
# 1. 開啟 Android Studio
npm run cap:android

# 2. 啟動模擬器（選擇 Pixel 5 或更高）
# 3. 運行應用
# 4. 準備好畫面後按 Ctrl + S 截圖

# 截圖會自動儲存到桌面
```

**使用實體手機：**
```
1. 連接手機到電腦
2. 啟用 USB 偵錯
3. 運行應用
4. 按 音量下 + 電源鍵 同時截圖
5. 從手機傳輸到電腦
```

## 🎨 截圖美化

### 使用 Figma（免費，推薦）

1. 註冊 Figma 帳號（免費）
2. 創建新設計（1080x1920）
3. 搜尋「phone mockup」找免費外框
4. 放入截圖
5. 新增標題文字
6. 匯出 PNG

### 使用 Canva（更簡單）

1. 選擇「Instagram 限時動態」模板（1080x1920）
2. 上傳截圖
3. 新增手機外框和文字
4. 下載

## 💡 提示

### 準備測試數據
- 3-4 個心智圖（各 10-20 個節點）

- 使用同一個測試帳號
- 統一語言（建議繁中）
- 統一主題（建議亮色）
- 電池顯示 90%+
- 時間顯示 10:30 左右
