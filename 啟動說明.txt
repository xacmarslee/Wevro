【最簡單的啟動方式】

1. 雙擊檔案：LAUNCH.bat

2. 視窗會保持開啟，等待看到：
   "serving on port 5000"

3. 開啟瀏覽器訪問：
   http://localhost:5000

═══════════════════════════════════════════════════

【如果 LAUNCH.bat 還是瞬間關閉】

使用 PowerShell 手動啟動（最可靠）：

步驟 1：按 Win + X，選擇「Windows PowerShell」

步驟 2：複製以下所有指令（全選 > 複製）：

───────────────────────────────────────────────────
cd E:\Projects\Wevro
npx tsx server/index.ts
───────────────────────────────────────────────────

步驟 3：在 PowerShell 視窗中右鍵貼上

步驟 4：按 Enter 執行

步驟 5：等待看到 "serving on port 5000"（約 15-30 秒）

步驟 6：開啟瀏覽器訪問 http://localhost:5000

═══════════════════════════════════════════════════

【停止伺服器】

在終端機視窗按：Ctrl + C

═══════════════════════════════════════════════════

【故障排除】

問題 1：視窗顯示錯誤後關閉
→ 用 PowerShell 方式啟動，可以看到完整錯誤訊息

問題 2：網頁顯示 "無法連線"
→ 確認終端機顯示 "serving on port 5000"
→ 確認沒有其他程式占用 5000 端口

問題 3：看到 "DATABASE_URL not set" 錯誤
→ 檢查 .env 檔案是否存在
→ 執行：notepad .env 檢查內容
