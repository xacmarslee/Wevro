# Wevro 應用截圖指南

## 📸 需要準備的截圖

### Google Play（Android）
- **數量：** 2-8 張（建議 6 張）
- **尺寸：** 至少 1080x1920 (9:16)
- **格式：** PNG 或 JPG

### App Store（iOS）
需要三種尺寸：
- **6.7" iPhone：** 1290x2796（必需）
- **6.5" iPhone：** 1242x2688（必需）
- **5.5" iPhone：** 1242x2208（必需）

---

## 🎯 建議的 6 張截圖內容

### 截圖 1：歡迎/登入畫面 ✨
**目的：** 展示專業的首次印象

**拍攝步驟：**
1. 開啟應用（登出狀態）
2. 確保顯示登入/註冊選項
3. 確保 Logo 和標語清晰可見

**文案疊加（可選）：**
```
AI 英文學習革命
心智圖 + 智慧字卡
```

---

### 截圖 2：心智圖編輯器 🧠
**目的：** 展示核心功能 - 視覺化學習

**拍攝步驟：**
1. 建立一個心智圖（例如：happy）
2. 展開 2-3 層（同義詞、反義詞、衍生詞）
3. 確保節點排列整齊美觀
4. 建議用亮色主題拍攝

**範例單字：**
- happy → joyful, cheerful, delighted
- run → sprint, jog, dash
- beautiful → gorgeous, stunning, lovely

**文案疊加：**
```
視覺化記憶
效果提升 3 倍
```

---

### 截圖 3：字卡列表 📚
**目的：** 展示字卡組織功能

**拍攝步驟：**
1. 建立 3-4 個字卡組
2. 每個字卡組命名清楚（例如：TOEFL 必備、商業英文）
3. 顯示字卡數量

**文案疊加：**
```
智慧字卡組
科學化管理
```

---

### 截圖 4：字卡練習 🎴
**目的：** 展示學習功能

**拍攝步驟：**
1. 開始字卡練習
2. 拍攝顯示單字卡片的畫面
3. 確保顯示「已知/未知」按鈕

**範例字卡：**
```
正面：exquisite
背面：adj. 精緻的；優美的

例句：The restaurant served exquisite French cuisine.
餐廳提供精緻的法國料理。
```

**文案疊加：**
```
間隔重複系統
記憶更持久
```

---

### 截圖 5：例句生成 💬
**目的：** 展示 AI 功能

**拍攝步驟：**
1. 查詢一個單字的例句（例如：accomplish）
2. 確保顯示多個例句
3. 展示中英對照

**文案疊加：**
```
AI 即時生成
自然真實的例句
```

---

### 截圖 6：同義詞比較 🔍
**目的：** 展示進階 AI 功能

**拍攝步驟：**
1. 查詢同義詞比較（例如：happy 的同義詞）
2. 顯示 3-5 個同義詞
3. 展示相似度評分和差異說明

**文案疊加：**
```
深度分析差異

```

## 🎨 截圖美化建議

### 方法 1：使用 Figma（推薦）

1. **創建 Frame**
   ```
   尺寸：1242x2688 (iPhone)
   或 1080x1920 (Android)
   ```

2. **添加設備外框**
   - 搜尋免費的 iPhone/Android 外框
   - 將截圖放入外框中

3. **添加文案**
   - 字體：思源黑體或 Noto Sans
   - 大小：48-72pt
   - 顏色：白色（加黑色陰影）
   - 位置：上方或下方

4. **添加裝飾元素**
   - 簡單的圖標
   - 漸變背景
   - 強調標記

5. **匯出**
   - 格式：PNG
   - 品質：2x 或 3x

### 方法 2：使用線上工具

**推薦工具：**
- [Previewed](https://previewed.app/) - 免費設備外框
- [Mockuphone](https://mockuphone.com/) - 簡單的外框生成
- [Canva](https://www.canva.com/) - 添加文字和裝飾

## 📱 如何拍攝截圖

### Android

**使用模擬器（推薦）：**
```bash
# 1. 開啟 Android Studio
npm run cap:android

# 2. 啟動模擬器（選擇 Pixel 5 或更高）
# 3. 運行應用
# 4. 使用模擬器工具列的相機圖標截圖
#    或按 Ctrl + S (Windows) / Cmd + S (Mac)

# 截圖位置：
# Windows: C:\Users\你的用戶名\Desktop\
# Mac: ~/Desktop/
```

**使用實體手機：**
```
1. 連接手機到電腦
2. 啟用 USB 偵錯
3. 運行應用
4. 按 音量下 + 電源鍵 同時截圖
5. 從手機傳輸到電腦
```

### iOS

**使用模擬器：**
```bash
# 1. 開啟 Xcode
npm run cap:ios

# 2. 選擇模擬器：
#    - iPhone 14 Pro Max (6.7")
#    - iPhone 14 Plus (6.7")
#    - iPhone 11 Pro Max (6.5")
#    - iPhone 8 Plus (5.5")

# 3. 運行應用
# 4. 截圖：Cmd + S

# 截圖位置：~/Desktop/
```

**使用實體 iPhone：**
```
1. 連接 iPhone 到 Mac
2. 運行應用
3. 截圖：
   - iPhone X 以後：音量上 + 側邊按鈕
   - iPhone 8 以前：Home 鍵 + 電源鍵
4. 從照片 App 傳輸到電腦
```

## 💡 額外提示

### 準備測試數據

在拍攝前準備好：
- 3-4 個精心設計的心智圖
- 4-5 個命名清楚的字卡組
- 每個字卡組 20-50 張卡片
- 查詢過的例句和同義詞

### 拍攝時機

- 選擇光線充足的畫面
- 避免拍攝載入中或錯誤畫面
- 確保電池顯示充足（90%+）
- 時間顯示合理（例如 10:30）

### 一致性

- 使用相同的測試帳號
- 統一的語言設定（建議繁中）
- 統一的主題（亮色或暗色）
- 統一的字體大小
